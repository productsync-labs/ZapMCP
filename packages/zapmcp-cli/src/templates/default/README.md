# Project Name

A ZapMCP project for interacting with AI models.

## Getting Started

```bash
npm install
npm run start
```

This will start your ZapMCP server at http://localhost:3000/mcp

## Features

- FastMCP server with SSE transport
- Custom tools for AI interaction
- Easy configuration
