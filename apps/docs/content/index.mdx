# Welcome to ZapMCP

ZapMCP is a powerful framework for building Model Context Protocol (MCP) applications.

## Getting Started

To get started with ZapMCP, you can create a new project using our CLI:

```bash
npm create zapmcp@latest my-project
```

## Features

- Easy to use API
- Type-safe development
- Built-in tools and resources
- Flexible configuration options
